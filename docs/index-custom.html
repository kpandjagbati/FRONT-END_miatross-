Génération de la Documentation

Pour régénérer la documentation après modifications du code:

npm run docs

Cette commande:

  • Scanne tous les commentaires JSDoc
  • Génère une documentation HTML statique
  • Place les fichiers dans le dossier docs/

Standards de Documentation JSDoc Utilisés

  • @fileoverview - Vue d'ensemble du fichier
  • @component - Indique un composant React
  • @param - Documenta les paramètres
  • @returns - Type de retour
  • @description - Description détaillée
  • @example - Exemples d'utilisation
  • @version - Numéro de version
